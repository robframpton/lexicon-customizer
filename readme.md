# Lexicon Customizer

> A desktop app for customizing lexicon look and feel that integrates with Liferay themes

## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
